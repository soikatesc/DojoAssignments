JavaScript Basics

Javascript is a cross platform object oriented programming. Javascript is extremely popular variety of reasons. It is a small and lightweight language allowing maximum flexibility for developer for take it into bunch of differnt directions.


  • Variable Declaration: Javascript variables are containers for storing data values-imagine a cup you fill with coffee, the cup holds the coffee, a variables holds the value. All javascript variables must be identified with unique names. The unique names are called identifiers. var x;
  • Variable Assignment:Assignment operators assign values to JavaScript variables - our cup can now have coffee poured in it, giving our variaable a value to hold. The = assignment operatior assigns a value to a bariable. var x=10;
  • Data Types: Data types are an importatn concept; to be able to operate on variables you need to know the data type. There are six data types that are JavaScript primitives: Boolean - true or false; null - null aka nothing; Number - 42 or 3.1416; String - "Coding Dojo Rocks!"; Array - [1,'Coding',2,'Dojo']; and Object - {first_name:'Jane', last_name:'Doe'}
  • Conditionals: when you write code, you want to perform differnt actions for differnt desicisions - hitting different code blocks based on values or conditions that have ben met. You can use conditional statements in your code to accomplish this. There are the following conditional statements: if a specified condition is true, do this code in our code block; else if to specify a new condition to test, if the first condition is false; we execute this block of code;
  • Loops: There are many differnt kinds of loops in every programming language, but they all essentially do the same thing: they will repeat an action some number of times. Imagine you have to run a mile, well you run around the track four times and then you stop. Thats a loop!
  • Functions: Functins are an excaspulation of a code blcok. When we call our function this will run that code block. Think of it as a list of instructions. As an example imagine we are putting together a desk from Ikea, we open up the instruction manual and get started, first we screw the legs to the table top; next we place the table the right way up. Done! We finished our instruction. Sadly ther are a ton more tables to do so lets call our function over and over and over again.
